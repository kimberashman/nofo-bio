# Contributing to NOFO.bio

NOFO.bio is a curated list. Entries are added and verified by the maintainer, not crowd-sourced. The best way to help is to flag things — additions, corrections, dead links.

## How to flag something

- **Missing opportunity?** [Open an issue](../../issues/new) titled `Add: <name>` with a link to the funder's page and a one-line scope.
- **Wrong or out-of-date deadline?** [Open an issue](../../issues/new) titled `Update: <name>` with the correction and a source link.
- **Or email** hello@nofo.bio — whichever's faster.

## Out of scope

- Anything that can't be verified against a funder's source page.
- Loans, equity-only investments without an open application process, and one-off press-release "challenges" with no application page.
